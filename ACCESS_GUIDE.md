## 🚀 启动命令

### 方式1: 使用配置文件 (推荐)
```bash
npm run dev:external
```

### 方式2: 直接启动
```bash
npm run dev
# 或
vite --host 0.0.0.0
```

### 方式3: 后台启动
```bash
nohup npm run dev:external > /tmp/vite.log 2>&1 &
```

## 🔧 配置信息

### Vite配置 (vite.config.ts)
```typescript
server: {
  port: 3000,
  host: '0.0.0.0', // 允许外部访问
  open: false, // 不自动打开浏览器
}
```

## 🛑 停止服务

```bash
# 查找进程
ps aux | grep vite

# 停止进程
kill <进程ID>

# 或使用端口停止
lsof -ti:3000 | xargs kill -9
```
